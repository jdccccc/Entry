# Entry

## 逻辑说明（极简）
- 启动：加载配置 → 开启 raw 模式 + 备用屏幕 → 进入主循环
- 主循环：
  - 主菜单：选择 TODO / Cyber / Bill，或退出
  - TODO / Cyber：读取配置中的文件 → 解析 Markdown 表格 → 通用表格组件渲染 → `e` 编辑，`r` 重新载入
  - Bill：
    1. `config.toml` 中通过 `bill_dir_path` 指定账单目录（默认 `tmp`），目录下放微信支付 `.xlsx`
    2. 进入页面显示待分析账单与可导出的报表数量
    3. `a` 分析：读取每个未分析账单，提取第二行昵称生成 `[昵称].md`；支出/收入按金额降序列出，统计 `<100` 的支出、总支出、总收入
    4. `o` 导出：可输入目标目录（回车沿用默认账单目录），写出 Markdown 报表
    5. `r` 仅刷新目录变化；`q` 返回主菜单
- 退出：恢复终端模式并离开备用屏幕
